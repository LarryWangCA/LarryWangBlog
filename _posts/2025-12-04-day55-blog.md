# C++ Tips 不同数据类型的范围
推荐使用long long防止溢出。  
```
| 类型        | 位数 | 范围（含端点）                                 | 备注           |
|-------------|------|-----------------------------------------------|----------------|
| int         | 32   | -2,147,483,648 ～ 2,147,483,647               | 最常用，会溢出 |
| long        | 32 或 64 | Windows: 32 bit；Linux/macOS: 64 bit    | ⚠ 不推荐跨平台 |
| long long   | 64   | -9,223,372,036,854,775,808 ～ 9,223,372,036,854,775,807 | 面试最安全 |
| uint32_t    | 32   | 0 ～ 4,294,967,295                             | 无符号         |
| uint64_t    | 64   | 0 ～ 18,446,744,073,709,551,615                | 无符号         |
```


## Leetcode 69 Sqrt(x)
使用二分法进行查找，左端为2，右端为num。  
注意更新left和right的边界（不能等于mid），容易造成死循环！
e.g. left=2,right=3, mid=2(因为整除向下取整);
需要更新为mid+1或者mid-1.
