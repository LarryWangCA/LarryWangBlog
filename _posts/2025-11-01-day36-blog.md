# C++ Tips
传统上max()和min()函数（#include<algorithm>）只能比较两个参数，支持多个参数求最值有两个方法：
1）嵌套写：int x = min(a, min(b, c));
2）使用初始化列表c++11以上：int y = max({a, b, c, d});

## Leetcode 188 买卖股票的最佳时机IV
本题本质上和LC123没有区别，只是123是最多买入卖出两次，本题为k次；
123属于手动列出了0，1,2,3,4五种状态，本题用j(j=2*k+1)代替即可。

## Leetcode 309 买卖股票的最佳时机含冷冻期（反复练习）
本题比之前的题目多了一个状态，因为加上了冷冻期，我们需要将不持有状态拆分为当天卖出和不持有（非卖出那天）。
dp[i][j]，第i天状态为j，所剩的最多现金为dp[i][j]。
状态0：买入、持有：
dp[i][0] = max(dp[i - 1][0], dp[i - 1][3], dp[i - 1][1] - prices[i]);
状态1：不持有
dp[i][1] = max(dp[i - 1][1], dp[i - 1][3]);
状态2：今天卖出
dp[i][2] = dp[i - 1][0] + prices[i];
状态3：冷冻期
dp[i][3] = dp[i - 1][2];
初始化：
dp[0][0] = -prices[0];
其余状态均为0//带入递推公式推导，或者理解为当天买入并卖出了，剩余现金为0。
从左向右遍历即可。
