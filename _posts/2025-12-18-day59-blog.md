# C++ Tips
sort适用于可以随机访问的迭代器 == “我能随意跳到第 i 个元素，并用 < 比较它们”  
注意，随机访问是按照位置访问，哈希查找也是O(1), 但不属于按位置访问,不能用sort。  
sort支持vector, deque和原生数组。

STL 中，顺序容器如 vector 使用 push_back 追加元素；容器适配器如 queue、stack 使用 push；  
而关联容器和链式容器由于不支持随机访问，通常通过 insert 或 emplace 来添加元素，其中 emplace 可以原地构造，效率更高。  


## Leetcode 5. Longest Palindromic Substring (反复练习) - 数组/dp类
暴力解法for循环嵌套列出所有子数组，每个单独判断是否回文，会超时。  
本题和之前代码随想录的一题很像，最直观的解法是中心扩散法。
for循环选择每个元素当做中心，然后向左右扩散，注意中心可以是奇数或者偶数（aba，abba),只用奇数会漏掉abba这种情况。  
```
    int expand(const string& s, int l, int r) {
        while (l >= 0 && r < (int)s.size() && s[l] == s[r]) {
            l--; r++;
        }
        return r - l - 1; // 注意：退出时多走了一步
    }
```
上述函数会多走一步，回文长度 = (r-1) - (l+1) + 1 = r - l - 1。

## Leetcode 252 Meeting Rooms (反复练习) - 数组
题目含义：如果不能参加所有会议 == 至少有两个会议时间有冲突。
暴力解法：for循环嵌套比较所有两两会议，maxStartTime < minStopTime即为有冲突。  
优化解法为sort排序后进行两两比较。从N^2 -> NlogN.  

## Leetcode 253 Meeting Rooms II （反复练习） - heap
题意：希望“从早到晚”依次安排会议，每来一个新会议，只要看当前最早结束的那个房间能不能腾出来。  
本题需要位数一个小顶堆   

  •	堆顶（最小 end）= 最早会空出来的房间  
	•	如果这个 end ≤ 新会议 start：说明房间已经空了，可以复用  
	•	否则：复用不了，需要新开一个房间  
最后堆的大小即为结果。
